# 🌤 Weatherly — Weather Website

## 🔗 Live Demo
https://deepikas-dev.github.io/weather-infomation/

## 📌 About

A responsive **weather website** built using HTML, CSS, and JavaScript that shows real-time weather data for any city.

## ✨ Features

* 🔍 Search weather by city
* 🌡 Temperature, humidity, wind, visibility
* 🌅 Sunrise & sunset
* 🎨 Dynamic background
* 📱 Responsive design

## 🛠 Tech Stack

HTML • CSS • JavaScript • OpenWeather API
